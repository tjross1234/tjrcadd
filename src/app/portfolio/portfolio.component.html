A sample of web applications created by us:

Manage Training Completions

This application allows administrative users to edit details of training completion entries in the database.

  • login
  • form to enter search criteria
  • displays entries matching the filter criteria with buttons to edit data or delete entry
  • modal dialog to modify entry data

Training Check

This application allows individuals to check the training completions for a given individual. Data is live from the database and can be used anywhere to check qualifications in real-time.

  • different user types
  • form to enter search criteria
  • displays individuals that meet search criteria
  • clicking on an individual lists training details
